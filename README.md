# LaTeX-beginner-starter



## Official Documentation

The <a href="https://www.learnlatex.org">LaTeX Documentation</a> is site is run by a small team of volunteers, with the LaTeX Project taking the lead. We are very happy to take contributions, provided they fall within the 'beginners' scope. We know there are lots of things we do not cover, but we've tried to agree a core curriculum that delivers for everyone <a href="https://github.com/learnlatex/learnlatex.github.io">Official Repository</a>.
